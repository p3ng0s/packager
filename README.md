P4CK4G3R
========
Get all of the repository and dependencies for p3ng0s and packages them for install

## Usage

```
#[p4p1@packager main/]$ ./setup.sh -h
Usage:
./setup.sh -a -> All Tools.
./setup.sh -b -> Build only.
./setup.sh -c -> Clean
./setup.sh -s -> Skip the AUR
Examples:
./setup.sh
./setup.sh -a
./setup.sh -b
./setup.sh -c
./setup.sh -s -b

```

To do a basic build do `./setup.sh`
